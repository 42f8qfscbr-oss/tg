# Python OpenCode Task Project

This folder is an independent Python task project for OpenCode automation.

Each project owns its own `.venv`.

## Setup

```powershell
.\scripts\python-bootstrap.ps1
```

## Start OpenCode

```powershell
opencode
```

For local DeepSeek V4 Pro after starting vLLM/SGLang:

```powershell
.\scripts\start-opencode-local-deepseek.ps1
```

## Validate

```powershell
.\scripts\python-validate.ps1
```
